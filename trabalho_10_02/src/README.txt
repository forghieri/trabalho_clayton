Projeto Java com Menu

Este projeto contém um menu principal que permite escolher entre as seguintes funcionalidades:

1. Calcular área, perímetro e diagonal de um retângulo.
2. Gerenciar dados de funcionários (salário líquido e aumento de salário).
3. Verificar nota final de alunos e aprovação.
4. Converter dólares para reais, considerando IOF.

Como executar:
1. Compile e execute o arquivo Main.java (comando dentro do console - javac Main.java                                              
java Main)

2. Escolha uma opção no menu para acessar a funcionalidade desejada.
